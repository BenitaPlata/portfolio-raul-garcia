# Instrucciones para el CV en PDF

Coloca aquí el archivo PDF del CV con el nombre exacto:

    cv-raul-garcia.pdf

El botón de descarga ya está conectado a esta ruta (`/cv-raul-garcia.pdf`).

## Cómo preparar el PDF

Para máxima compatibilidad con bases de datos de centros educativos:
- Formato A4
- Márgenes normales (2cm)
- Fuente legible (mínimo 11pt)
- Sin contraseña ni restricciones de impresión
- Nombre de archivo: `cv-raul-garcia.pdf` (sin espacios ni tildes)

## Hasta que tengas el PDF listo

El botón aparece en el portfolio pero simplemente no descargará nada hasta
que coloques el archivo aquí. No da error visible al usuario.
